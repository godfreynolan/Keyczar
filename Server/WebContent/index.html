Welcome to the Example RSA Decryption Server


This API offers the following endpoint:

  • Decrypt: /do/decrypt/havepublic?text=[encrypted string]

Sample URL:

  • Have encrypted string: http://betterdriving.riis.com:8080/RSAServer/do/decrypt/havepublic?text=[encrypted string]

This API is REST-based and returns JSON data.

Decryption Response Format:

{
  "result": "[decrypted text]"
}

Note: This API is subject to change at any time and is only provided as an example.
